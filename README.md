# BIWD2210 Web Technology Assignment

This project is a 5-page **News Portal** website built with HTML and CSS for the BIWD2210 individual assignment.

## Project Pages
- `index.html` - Home page with featured news and trending stories
- `categories.html` - News categories page with ordered and unordered lists
- `article.html` - Individual news article page
- `multimedia.html` - Multimedia page with embedded video, audio, and image gallery
- `contact.html` - Contact page with editor details and a form

## Rubric Coverage
- Semantic HTML elements and clean structure
- External stylesheet (`styles.css`)
- Responsive layout with Flexbox and Grid
- Functional navigation menu across all pages
- Internal and external hyperlinks
- Relevant images with captions
- Embedded media (video + audio)
- Ordered and unordered list usage
- AI RSS digest (multi-source headlines) with auto-generated highlights
- Real journalist names with links (Lesotho, South Africa, and Africa)

## Run Locally
1. Open the folder in your editor.
2. Open `index.html` in a browser.

## Deploy to GitHub Pages
1. Create a new GitHub repository.
2. Upload all files in this folder.
3. Go to `Settings > Pages`.
4. Under `Build and deployment`, set source to `Deploy from a branch`.
5. Select branch `main` and folder `/ (root)`.
6. Save and wait for GitHub to publish.
7. Copy the live URL shown in Pages settings.

## Deploy to Netlify
1. Log in to [Netlify](https://www.netlify.com/).
2. Click `Add new site` > `Import an existing project`.
3. Connect your GitHub repository.
4. Build command: leave empty, publish directory: `/`.
5. Deploy and copy the generated live URL.
